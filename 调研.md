### 基本界面

F型，左右结构，“后台管理系统布局” (Admin Dashboard Layout) 或 “侧边栏导航布局” (Sidebar Navigation Layout)。

- logo区
- 快捷操作：如 "新建知识库" 按钮，提供高频操作的入口。

- 用户/底部栏：显示当前登录用户（admin）的信息，包含头像和设置菜单。

### 知识库模块

- 知识库/个数：问答型、文档型
- 文档分类/目录结构
- 搜索功能：关键词
- 筛选功能：按照文件格式

**知识库类型与内容来源（系统逻辑）**

- **文档型知识库**：以「文档」为内容单位。支持**上传文件**（PDF、MD、HTML 等）→ 解析、分块、向量化；库内关键词搜索检索文档片段。
- **问答型知识库**：以「问答对」为内容单位。支持**新建问答**、**导入 CSV 文件**（列：question、answer、similar_questions、tags）；不支持上传普通文档，因为内容形态是结构化问答。库内关键词搜索检索问答。

文档型：
- 上传文件 → 文档列表 → 学习（解析/分块/向量化）→ 搜索

问答型：
- 新建 FAQ / 导入 CSV → 问答列表 → 搜索
