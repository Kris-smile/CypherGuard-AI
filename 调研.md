## web系统设计
### 基本界面
F型，左右结构，“后台管理系统布局” (Admin Dashboard Layout) 或 “侧边栏导航布局” (Sidebar Navigation Layout)。

+ logo区
+ 快捷操作：如 "新建知识库" 按钮，提供高频操作的入口。
+ 菜单导航：包含 "知识库"、"智能体"、"对话" 等功能模块。
+ 可收起/折叠：系统允许点击底部或顶部的按钮将此栏收起为图标模式，以腾出更多空间给右侧。
+ 用户/底部栏：显示当前登录用户（admin）的信息，包含头像和设置菜单。
